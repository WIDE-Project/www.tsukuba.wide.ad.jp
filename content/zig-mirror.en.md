WIDE Tsukuba NOC provides a mirror service for the [Zig programming language](https://ziglang.org/).

Domain name: `zig.tsukuba.wide.ad.jp`

## Software Used

At present (as of 2026/02/27), the mirror is implemented using the open-source software [go-mirror-zig](https://github.com/SavaLione/go-mirror-zig).

We would like to thank Savelii Pototskii, the author of this software.
